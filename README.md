# App-Agent

Application fullstack pour CODE-TALENT avec backend NestJS et frontend Next.js.

## Structure du Projet

- **Backend_APP_AGENT/** - API backend NestJS
- **Frontend_APP_AGENT/** - Interface utilisateur Next.js

## Installation

### Backend
```bash
cd Backend_APP_AGENT
npm install
npm run start:dev
```

### Frontend
```bash
cd Frontend_APP_AGENT
pnpm install
pnpm dev
```

## Déploiement

Instructions de déploiement pour le serveur CODE-TALENT sur Infomaniak.

## Technologies Utilisées

- **Backend**: NestJS, TypeScript
- **Frontend**: Next.js, React, TypeScript
